# What's ThreeDify?
ThreeDify is a online platform where you can upload images and create a 3D reconstruction of the images.

## Installation
1. Make sure you have `node-v13.11.0` and `yarn-v1.22.4`
2. Install dependencies
```
$ yarn
```

## Build
```bash
$ yarn build
```

## Run
```bash
$ yarn start
```

## Lint
Check lint errors
```bash
$ yarn lint
```
Fix lint errors
```bash
$ yarn lint:fix
```
